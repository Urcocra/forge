# FORGE Attachments

This directory contains supplementary materials accompanying the FORGE benchmark paper.  
All files are provided to support **reproducibility, verification, and falsification** of the reported results.

No new claims, conclusions, or evaluation criteria are introduced in these attachments.

---

## Purpose and Reading Guide

The attachments are intentionally modular. Reviewers and readers may consult only the sections relevant to their questions.

- For **quick reproducibility checks**, see `reproducibility.md`
- For the **full conceptual definition** of failure layers, see `failure-layer-taxonomy.md`
- For **statistical methodology and uncertainty**, see `stats-details.md`
- For **ethical and transparency scope**, see `esg-disclaimer.md`
- For **model-specific behavior details**, see `micro-reports/`

---

## Summary

The main paper states the claims.  
The attachments show how those claims can be checked.
